# InPTA.DR2
The second official data release of the Indian Pulsar Timing Array including dataset of 27 millisecond pulsars (MSPs) from Band 3 (300-500 MHz) and Band 5 (1260-1460 MHz) of the upgraded Giant Metrewave Radio Telescope (uGMRT).
This dataset consists of:
1. Time of Arrivals (ToAs, .tim file) and pulsar timing model (.par file).
2. Dispersion Measure (DM) time series.
3. Noise-free templates used for cross-correlation.
4. Scripts used for analysis.
